# Drug-Repurposing-PrimeKG-Capstone-2026
This project leverages the Precision Medicine Knowledge Graph (PrimeKG) to analyze multi-level biomedical relationships between drugs, diseases, genes, pathways, and phenotypes, and realise potential drug disease associations.
